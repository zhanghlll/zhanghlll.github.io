github建网站过程

发表于 2023-05-16 更新于 2023-05-23

github建网站过程

1.新建文章

hexo new "文章名"

2.上传文章

hexo d

3.启动本地服务端

出现 “Hexo is running at http://localhost:4000 . Press Ctrl+C to stop.” 代表启动成功

4.配置博客

重点是这几行

# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: git@github.com:zhanghlll/zhanghlll.github.io.git
  branch: main

美化（选做）

访问 https://hexo.io/themes 挑选喜爱的主题，作者一般会提供安装引导，不在此赘述。

5.托管站点

  • 在github上建立私有仓库
  • 在 Hexo 根目录执行（其中 $repository_url$ 替换为你实际的私有仓库 URL）

git init
git add .
git commit -m "init"
git branch -M master
git remote add origin $repository_url$
git push -u origin master
